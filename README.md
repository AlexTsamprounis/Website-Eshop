# AT.COLLECTION – Dynamic eShop  
MSc ICT – Ανάπτυξη Εφαρμογών στο Διαδίκτυο (Mscict114)

## 📌 Περιγραφή Έργου

Το παρόν project υλοποιήθηκε στο πλαίσιο του μαθήματος  
«Ανάπτυξη Εφαρμογών στο Διαδίκτυο» του ΠΜΣ «Πληροφορική και Εφαρμογές»  
του Πανεπιστημίου Δυτικής Αττικής.

Αντικείμενο της εργασίας είναι η ανάπτυξη ενός δυναμικού ηλεκτρονικού καταστήματος (eShop) 
με χρήση HTML, CSS, JavaScript και PHP, καθώς και σχεσιακής βάσης δεδομένων MySQL.

Το σύστημα υποστηρίζει:
- Περιήγηση επισκέπτη
- Εγγραφή και σύνδεση χρήστη (authentication)
- Διαχείριση καλαθιού
- Ολοκλήρωση αγοράς (checkout simulation)
- Αποθήκευση δεδομένων σε βάση

## ⚙️ Λειτουργικότητες Συστήματος

### 🏠 Αρχική Σελίδα
- Τίτλος και περιγραφή καταστήματος
- Μενού πλοήγησης
- Σύνδεση με βασικές ενότητες

### 🛍 Σελίδα eShop
- Δυναμική εμφάνιση προϊόντων από βάση δεδομένων
- Κατηγοριοποίηση προϊόντων
- Προβολή:
  - Εικόνας
  - Τίτλου
  - Περιγραφής
  - Τιμής
- Κουμπί "Add to Cart"

### 🛒 Σελίδα Καλαθιού
- Προβολή προϊόντων καλαθιού
- Αλλαγή ποσότητας
- Αφαίρεση προϊόντος
- Υπολογισμός συνολικού κόστους
- Μεταφορά σε Checkout

### 💳 Checkout
- Πρόσβαση μόνο για διαπιστευμένους χρήστες
- Εισαγωγή στοιχείων κάρτας (simulation)
- Μετατροπή καλαθιού σε παραγγελία
- Καταχώρηση αγοράς στη βάση δεδομένων

### 👤 Authentication
- Register
- Login
- Logout
- Χρήση PHP Sessions
- Έλεγχος πρόσβασης σε προστατευμένες σελίδες

## 🗄 Δομή Βάσης Δεδομένων

Η βάση δεδομένων περιλαμβάνει πίνακες όπως:

- users
- products
- categories
- cart
- orders
- order_items

Τα δεδομένα αποθηκεύονται σε MySQL σύμφωνα με τις απαιτήσεις της εργασίας.

## 🚀 Οδηγίες Εκτέλεσης (Local)

1. Εγκαταστήστε XAMPP ή MAMP.
2. Τοποθετήστε τον φάκελο του project στο:
   - htdocs
3. Δημιουργήστε βάση δεδομένων (π.χ. at_collection_db).
4. Κάντε import το αρχείο sql/at_collection_db.sql.
5. Εκτελέστε:
   http://localhost/Website-Eshop/index.php

🛡️ Ασφάλεια & Τεχνική Υλοποίηση
Δόθηκε ιδιαίτερη βαρύτητα στην ασφάλεια της εφαρμογής, εφαρμόζοντας τις εξής πρακτικές:

SQL Injection Prevention: Χρήση Prepared Statements για την ασφαλή αλληλεπίδραση με τη βάση δεδομένων.

Authentication & Sessions: Χρήση PHP Sessions για την προστασία ευαίσθητων σελίδων (Checkout, User Profile).

Responsive Web Design (RWD): Πλήρης προσαρμογή της διεπαφής σε όλες τις αναλύσεις οθονών.

🛠️ Τεχνολογικό Stack

•	Frontend: HTML5, CSS3, JavaScript.
•	Backend: PHP 8.x (Modular logic).
•	Βάση Δεδομένων: MySQL (Relational Schema).
•	Εργαλεία: Git/GitHub για Version Control, XAMPP περιβάλλον ανάπτυξης.


⚙️ Λειτουργικότητες
Δυναμικό eShop: Άντληση και προβολή προϊόντων (εικόνα, περιγραφή, τιμή) σε πραγματικό χρόνο από τη βάση.

Σύστημα Καλαθιού: Πλήρης διαχείριση (προσθήκη, αλλαγή ποσότητας, αφαίρεση) με δυναμικό υπολογισμό συνόλων.

Authentication: Σύστημα εγγραφής (Register), σύνδεσης (Login) και αποσύνδεσης (Logout).

Checkout Simulation: Προσομοίωση ολοκλήρωσης παραγγελίας και καταχώρηση στοιχείων στη βάση δεδομένων.
